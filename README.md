# Vela

Vela is an experimental programming language currently in its early stages.

The project is being built incrementally, starting with the core tooling required to read, parse, and execute simple `.vela` files.

## Development

Vela currently uses TypeScript and Bun for its initial implementation.

### Requirements

* [Bun](https://bun.sh)

### Commands

```bash
bun run dev
bun run typecheck
bun test
```

## Project structure

```text
src/        implementation source code
tests/      automated tests
examples/   example Vela programs
```

## Status

Vela is under active development and is not ready for production use.

The syntax, architecture, and tooling may change as the project evolves.
